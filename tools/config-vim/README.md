# VIM Configuration
* This is VIM configuration.
* This config is forked from [posquit0](https://github.com/posquit0/).

## Usage
```sh
$ cd ~
$ git clone https://github.com/iltaek/til/tree/master/tools/config-vim ~/.vim
$ ln -s .vim/vimrc .vimrc
# If you use NeoVIM
$ ln -s ~/.vim .config/nvim

# Plug-in install
$ vim
:PlugInstall
```
